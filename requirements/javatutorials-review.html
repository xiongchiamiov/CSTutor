1.5.2. Sun's Java Tutorials

The Java Tutorials are a collection of lessons put together by Sun Microsystems. They range in difficulty from the very basics of Java, through interactive GUIs, all the way to specialized tutorials such as Custom Networking. While the tutorials are purely web based, the introduction tutorial provides instruction to download free software in which you can run and edit their examples.

Good Features:

  • Tutorials contain examples in the the Java language as well as pictures and diagrams
  • Broken down into categories and sub lessons
  • Offers a tutorial for preparation to becoming Java Certified
  • Variety of topics ranging from basic to very in-depth
  • Very detailed explanations so a true beginner shouldn't become lost

Bad Features:

  • Bad navigation

Missing Features:

  • Tree View for easy navigation and jumping around
  • Quizzing
  • Code can not be edited or ran directly
