2.3.3 Using the Statistics Page

This scenario shows an instructor viewing various statistics relating to a class.

2.3.3.1 Viewing the Initial Screen

With no configuration performed in other sections of the software, the main pane on the statistics page is empty. The left pane has only a link to the home section, defined as the page viewed immediately after login.

Figure 2.3.3-1: Initial Screen

2.3.3.2 Viewing the Initial Screen, with Data

With lessons and assignments created, and users associated with the class, the initial screen will show instructors a table, displaying information about the amount of time spent on a particular page, as well as the score received, if the page in question contained quiz questions and the student has attempted to answer them.

Figure 2.3.3.2: Initial Screen, with Data

2.3.3.3 Viewing Statistics by User

From the initial screen, the instructor clicks the name of a student to view more detailed statistics on his or her performance. This is the default page that a student will see, once there are lessons added to the class.

Figure 2.3.3.3-1: User Statistics

2.3.3.4 Viewing Statistics by Lesson

From the initial screen, the instructor clicks the name of a lesson to view more detailed statistics on that lesson.

Figure 2.3.3.4-1: Lesson Statistics
