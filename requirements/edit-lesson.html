2.3.1.3 Editing a Class/Lesson

An instructor is taken to an Edit Lesson screen in two ways

1. The instructor has just created a new class or lesson and was taken this screen to fill in the main page. The initial screen in this case would be blank(For more information, see "Creating new Classes/Lessons").

2. The instructor was browsing his or her lessons and classes and clicked on the "Edit" button. The screen will be populated with the data for that lesson and will be editable by the instructor.

Figure 2.3.1.3-1
To edit an existing lesson, the instructor browses lessons in the navigation bar. Notice the selected lesson is highlighted. The data will show up as though the instructor was following the lessons. Once he or she finds the lesson, they first click on the topic to view it, then click on Edit to begin editing.


Preview/Save button: Clicking the Preview/Save button saves the current edits. The user is then taken to a screen that shows the lesson a student would view it. This does NOT make the changes visible to the students in the class.

Save button: Clicking the Save button save the current edits only. A save conformation will appear if save is completed.

Figure 2.3.1.3-2



Publish button: Clicking the Publish button save the current edits and publishes them to the class. Students will be able to view the changes after the lesson is published.

Figure 2.3.1.3-3



Import button: Clicking the Import button will import data into the lesson. Files that can be imported are: jpeg, gif, png, html, txt, java, c, cpp
Once the button is pressed, a file browser opens to select a file.

Figure 2.3.1.3-4



Options button: Clicking the Options button pushes a dropdown list of options onto the screen.

Figure 2.3.1.3-5


Modify Prerequisites: Clicking this button allows the instructor to pick from a list of previous quizzes. Selecting an item means that the Student cannot view the page without first completing the selected quiz with the indicated score. When selecting a quiz the Instructor may select a minimum score required. See Figure 2.3.1.3-6 below.

Figure 2.3.1.3-6



Delete Lesson: Deletes the current lesson and returns the Instructor to a nearby lesson or home page if there are none.

Undo All Changes: This removes all changes from the working copy by copying the currently published version of the lesson into the working copy.

Export Class: This button prompts the user for a save location then packages the class into an encrypted format and saves into the previous location with the title of the class as its name.

Import Class: This button prompts the user for a location then unpackages the saved class and loads it into CSTutor.

Toolbox button: Clicking the Toolbox button places a toolbox onto the screen. It can be removed by clicking Toolbox button again.

Figure 2.3.1.3-7


Below exists a walkthrough illustrating how to use the tools to create the lesson seen on this page.

After creating the lesson, CSTutor presents the user with a blank window. Clicking inside this window allows the user to see a cursor and type text into the window. First the user types the title of the lesson "Creating Arrays". Next the user decides to change the font of the title. To do this the user highlights the title then clicks the Font button.

Figure 2.3.1.3-8



Now the Instructor wishes to center the title of the page. To do this the Instructor highlights the title then clicks the Align button which spawns an alignment dialogue. Here the Instructor chooses the center button.

Figure 2.3.1.3-9



To finalize the title the Instructor makes the text larger by clicking the Size button. This button prompts the user for text size.

Figure 2.3.1.3-10



The Instructor types more text but then decides to put in a link at the end of the paragraph to the java sun array tutorial. To do this the user clicks the Link button which brings up a dialogue prompting the Instructor for the URL and the displayed text for the link.

Figure 2.3.1.3-11



Now the Instructor wants to put in a picture to help illustrate arrays. To do this the Instructor clicks the IMG button. This button brings up a dialogue prompting the user for the image location. If the Instructor wishes to center the picture he must click on the picture to select it, then use the Align button.

Figure 2.3.1.3-12



The Instructor finishes off the lesson by adding in a code box by clicking the Code button. This button places a code box where the Instructor's cursor is pointing. The code button creates the code box initially empty but the Instructor can fill it with text by clicking inside the code box and typing.

Figure 2.3.1.3-13



From here the Instructor may view from the Student's perspective by clicking Preview/Save or simply save his work by clicking Save. Once satisfied the Instructor can allow Students access to the page by clicking the Publish button.
