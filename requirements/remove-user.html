2.3.4.2 Deleting a Student from the Class Roster

This scenario shows the instructor of a class how to remove students from the class roster. Removing students is important because it allows the instructor to properly manage the class roster. In this scenario, we assume that the instructor has logged in, selected the appropriate class, and is viewing the roster page. Details of logging in as an instructor can be found in Section 2.1

To remove a student from the class roster, the instructor selects Delete User on the roster management screen and then the diaglog box shown in Figure 2.3.4.2-1 is displayed by the system.



Figure 2.3.4.2-1: Remove User Dialog.


The user field is a one line string where the student's name to be deleted is typed in. After the student's name is entered click Delete to delete the student from the class roster.
