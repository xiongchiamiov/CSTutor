2.5. Communication Tools

CSTutor is primarily intended to be used as a solitary study aid. However, the following methods of communication are integrated into CSTutor.

2.5.1 Class Chat

During any lesson, select the "Chat" Button (see 2.1 UI Overview) to open up the integrated IRC interface. Once the interface has loaded and connected, the interface behaves as a standard IRC chat room.

Figure 2.5-1: Web based Chat room Interface.



Each Class will have its own chat room.
