1. Introduction

This document defines requirements for an interactive electronic tutoring program called CSTutor. CSTutor allows users to learn programming lessons in an interactive manner. Lesson creators use tools within the program to create a hierarchical lesson design. Creators may place text boxes with example code into the lessons. These text boxes allow users to edit and run the code in order to better understand the language. CSTutor also provides functionality to create quizzes and to track user historical progress through quizzes and lessons. The program initially supports Java lessons but has functionality to import other languages.
