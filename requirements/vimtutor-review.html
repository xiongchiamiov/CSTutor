1.5.1. VimTutor

Vimtutor is a great first step to learning vim. It walks the user through the basics of the editor, including a couple of "wow, cool!" features to show how vim can improve productivity. At it's core, it's simply a text file, which is the environment where vim's features are strongest. Because the tutor is just a text file that you edit with vim, the system employs a "learn by doing" approach that drops you right into vim.

Good Features:

  • "Learn by Doing" approach. You have to learn the first few lessons just to get to the next one.
  • Shows off most useful features without bogging down in complex advanced topics
  • Exceptionally clear instructions. Effort to be concise and clear is evident.

Bad Features:

  • Learning curve is high. Not everyone will want to finish.
  • Not very flashy. ASCII text file only.
  • Installed with VIM, only contains a small set of lessons. Only extendable by editing the file locally or submitting a patch to the maintainer.
  • Additional "lessons" involve reading manpages or static documentation. No further interactive lessons.
  • "Retaking" lessons involves hitting the "undo" command multiple times, or restarting the entire lesson process.
  • All lessons on same page.

Missing Features:

  • Ability to add new lessons
