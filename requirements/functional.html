2. Functional Requirements

The primary functionality of the CS Tutor is to allow students to view lessons for a class that are created be an instructor. Instructors create lessons through the lesson management screen. In addition, instructors can create quizzes that can track the students' progress. The CS Tutor also gives them detailed information about how the class is progressing so he/she can indentify weaknesses.

Following an overview of the CS Tutor user interface, details of system use are presented in the following scenarios:

  • an individual user logging into the CS Tutor
  • an instructor trying to use the admin page
  • a student trying to browse a lesson
These scenarios cover all functional requirements for the system.
