1.5.6 Sophrinix Ruby Tutorial (Try Ruby!)


Sophrinix created an interactive Ruby tutorial that is programming based.  The page is centered around an interpreter and there is text at the bottom saying what chapters there are.  You choose from the chapters and a step by step tutorial begins.  Each step requires something to be typed to the interpreter.  Once you enter a correct command, you go onto the next section and it explains what the command did and how it can be used.


This is an extremely effective tutorial for Ruby.  Every step requires user interaction and the explanations are very clear.  The way the lessons are structured allows programming novices to follow along and understand while not boring the more experienced programmers.  It is not an extremely feature heavy program but is easily followed and keeps interest.


Good Features:

  • Builds skills from the ground up with extremely detailed tutorials
  • Not only allows, but requires code to be written to move on
  • Goes over both working code and common errors
  • Very easy on the eyes.  Graphics flow and are easy to understand.


Bad Features:

  • Navigation between chapters is unclear.  Chapters are called 1-8 and don't have meaningful names
  • Progress is not easily trackable.  Once a topic is learned, it's hard to go back.


Missing Features

  • Easy global navigation
  • User history
