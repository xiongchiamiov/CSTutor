2.3.2

The spec didnt go into much detail about data validation on quizzes. Here are the rules that are enforced. Quiz Title must be unique and not blank. All quesiton prompts must not be blank. No answer can be blank. Multiple choice questions must have a correct answer. QUestions must be in a valid order. Code quesiton expected output cannot be blank. Any prerequisites (multiple can be selected) must have a corresponding passing path, otherwise the quiz would be unreachable.
