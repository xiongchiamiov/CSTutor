The CSTutor software system provides functionality to view lessons created by an instructor, track stats of users, and take quizzes that are created by an instructor. The system is comprised of two application programs:

  • the CSTutor website, for regular users and instructors. Regular users can view lessons and take quizzes and Instructors can create lessons and quizzes. The website sends requests to the server for data.
  • the CSTutor Server, accepts requests from users and responds with HTML and generates pages for the website.
The requirements specification document has complete details of CSTutor functionality.
