For Loops

For loops are a type of loop that allow you to define a start expression, a test expression, and a count expression.

The start expression is executed once, before the loop body is is entered. The test expression is executed at the beginning of each run through the loop. If the expression is true, the loop body is run. If the expression is false, the loop exits. Finally, the Count expression is executed at the end of the loop body. See the code below: 

int x;
for (x = 0;  /* Start Expression */
     x < 30; /* test expression */
     x++) {  /* count expression */
   System.out.println(x);
}

In this case, x is set to 0, and the loop is run, each time incrementing x by 1. When the condition x < 30 is no longer true, the loop exits.
