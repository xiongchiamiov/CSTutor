What are Arrays

Arrays are a convenient way of grouping data together. For example, if you need to keep track of the number of people who came to my amusement park each day for a month, you could use an array of 31 integers represent this.

Declaring Arrays

Continuing with the amusement park example, without Arrays, you would need a separate integer for each day of the month like so: 

int day1, day2, day3, day4;
int day5, day6, day7, day8;
...

With arrays, we can represent all 31 days using a single identifier. 

int [] foo;
foo = new int[31];

We can access individual elementes of the array using the index operator. You should note that indexes begin at 0. The following code when ran will cause an ArrayIndexOutOfBoundsException. 

int [] foo;
foo = new int[31];
foo[0] = 153;
System.out.println(foo[0]);
foo[31] = 142;
System.out.println(foo[31]);
Exception in thread "main" java.lang.ArrayIndexOutOfBoundsException: 31
					at Test.main(Test.java:7)
